Architecture Technique · Nouveauté v2.4

Mémoire biologique des agents

Épisodique · half-life 7 j

Observations horodatées par session — erreurs détectées, patterns de code, décisions locales

Sémantique · half-life 30 j

Patterns consolidés par similarité Jaccard — savoir architectural inter-sessions

Décisions · half-life 21 j

Décisions architecturales persistantes — invalidées automatiquement par git log

  • Décroissance exponentielle — courbe d'Ebbinghaus, auto-élagage sans intervention
  • Renforcement à la lecture — chaque accès ajoute 2 j au half-life de l'entrée
  • Amplification des erreurs — erreurs mémorisées 2× plus longtemps
  • Niveaux de confiance — verified → observed → inferred → stale
  • Zéro dépendance — JSON pur, stocké dans .valora/memory/

Sécurité & Conformité

6 niveaux de protection

Credential Guard
Redaction des variables d'environnement, blocage des fichiers sensibles, scan des sorties
Command Guard
Bloque l'exfiltration, les patterns réseau dangereux, eval et accès distants
Détection d'Injection
Scan des résultats d'outils avec score de risque, quarantaine et redaction automatique

Mise en Route

De zéro à votre première PR en 6 commandes

terminal 
# 1. Installation globale
pnpm add -g @windagency/valora

# 2. Initialiser dans votre projet
cd mon-projet && valora init

# 3. Créer le plan d'implémentation
valora plan "Ajouter l'authentification OAuth"

# 4. Exécuter l'implémentation
valora implement

# 5. Révision + commit conventionnel
valora review-code && valora commit

# 6. Pull request automatique
valora create-pr
