Execution & Conditions
Execution Loop
async reduce() · Promise.all()
Groups run sequentially via reduce(). Within a parallel group, all stages run via Promise.all() concurrently
Synchronisation Barrier
Every group boundary
Promise.all() means all parallel lanes must complete before the next sequential stage can start — prevents reading partial results
Conditional Stages
conditional: $STAGE_*.key == value
Evaluated at runtime via variable resolver. Falsy → stage filtered out before group executes. Safe if required: false
Implicit Dependencies
$STAGE_<name>.<output_key>
No depends_on field. Dependencies declared as variable references. Missing outputs resolve to undefined — not validated at schedule time
